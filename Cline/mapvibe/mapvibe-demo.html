This demo shows how the MapVibe Jobs JavaScript works to display job listings.

How to Add This to Your WordPress Site

  1. Upload the mapvibe-jobs.js file to your WordPress theme directory or a custom plugin directory.
  2. Enqueue the script in your WordPress theme's functions.php file using the following code:
function enqueue_mapvibe_jobs_script() {
    wp_enqueue_script('mapvibe-jobs', get_template_directory_uri() . '/mapvibe-jobs.js', array(), '1.0.0', true);
}
add_action('wp_enqueue_scripts', 'enqueue_mapvibe_jobs_script');

Alternatively, you can add the script directly to your WordPress page using a custom HTML block or by editing your theme's footer.php file.
